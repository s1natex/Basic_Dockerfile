# How to use:
- Clone the repo:
```sh
git clone https://github.com/s1natex/Basic_Dockerfile
```
- Build the image:
```sh
docker build --build-arg USER_NAME="add-your-name" -t passgen .
```
- Run the image:
```sh
docker run -it passgen
